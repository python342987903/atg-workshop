# 六安阿堂哥网络工作室

专业的数字化服务网站，提供聚合支付、视频制作、修图服务、网站开发等服务。

## 功能特性

- 🏠 首页展示四大服务模块
- 👤 用户注册和登录
- 🛒 购物车功能
- 💳 在线支付（支持微信和支付宝）
- 📦 订单管理和物流查询
- 💬 在线客服（QQ机器人）
- 🎯 后台管理系统

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- LocalStorage (数据存储)

## 部署说明

### GitHub Pages 部署

1. 在 GitHub 上创建仓库
2. 上传文件到仓库
3. 进入 Settings → Pages
4. 选择 main 分支，点击 Save
5. 等待几分钟，网站就上线了！

### Vercel 部署

1. 访问 vercel.com
2. 导入 GitHub 仓库
3. 点击 Deploy
4. 立即上线！

### Netlify 部署

1. 访问 netlify.com
2. 直接拖拽文件夹到网页
3. 立即上线！
